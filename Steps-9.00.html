Instruction for the Flat File Connection:
The sound from my video got cut off so I want to add directions on how to create the destination for the null values:
1) Drag the flat file destination from "other destination" to the data flow. 

2) Connect the conditional split to each destination. NullreportId needs to be connected to the flat file. Conditional split default output needs to be connected to the OLEDB destination. 


3) Double click on the flat file then click new. 

4) Click on new and select the "delimited" option.

5) Rename the connection manager to NullFile. 

6) Create a flat file in your local file folder then click on browse to select that file. 

7) Make sure column names in the first data row in unchecked then press OK. 

8) Then it will take you back to the Flat File Destination Editor. Then click on mappings and then press ok.
